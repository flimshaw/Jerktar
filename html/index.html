I've spent my downtime today making scripts for my computer, because it suddenly dawned on me that I could use Dropbox to sync my .bash_history and .bash_profile and, consequently, paths and aliases and all kinds of stuff. I set it up on my work computer and my old MBP, so now they share a bash history. I've started trying to get a few scripts together to automate some of the command-line stuff I do every day, all day. This is good for a few reasons I think. 1 is, I do a LOT of different stuff on the command-line, and, I mean, real hackers don't seem to have any problem jumping around and knowing this stuff, and maybe it'll become second nature for me later, but I find that I have a really hard time switching between different programming languages or sets of tools. The other day, another dev at my office was asking me to do some simple MySQL stuff, but I hadn't touched the console in forever so a lot of it was gone. But now that I know how to use it, it's really powerful. The Django shell. Man, I underestimated that thing. So great for learning how python works and quickly testing different syntaxes or object types or whatever. For when json is being cranky. Which is always. MORE LIKE NERDTAR! AM I RIGHT?
